# convertErmLog2Csv
Converts ERM log.bin format to CSV format (for use with MegaLogViewer)

Note: This currently only works for M4C Ecu (KTM TBI models) log files generated by ERM.
This will convert all files in current directory from `log_<num>.bin` to `log_<nun>_m<map>.csv`

Captures following fields (converts degC to degF):
RPM
throttlePosition
crankPressure
ignitionAdvancepowerValve
coolantTemp_F
ambientTemp_F
ambientPressure_kPa
batteryVoltage
